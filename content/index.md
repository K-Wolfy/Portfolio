# Welcome to My Portfolio

This is a Quartz-powered portfolio site that will showcase my work and CV.

## About This Site

This site is built with [Quartz 4](https://quartz.jzhao.xyz/) and will be automatically updated from my Obsidian vault via GitHub Actions.

## Features

- **Automated Content Updates**: Content is synced from Obsidian vault
- **Modern Design**: Clean, responsive layout
- **Fast Loading**: Static site generation for optimal performance
- **Search & Navigation**: Built-in search and easy navigation

## Getting Started

The portfolio content will be populated automatically once your Obsidian vault starts pushing content to this repository.
